# Support Intake Template

사용자 문의를 받을 때 그대로 복사해서 채웁니다.

## 기본 정보

```text
접수일:
사용자 이름:
연락처:
사용 주소: https://app.getvibeshare.com
휴대폰 모델:
휴대폰 OS:
PC 브라우저:
```

## 문제 유형

- [ ] 웹앱 접속
- [ ] QR 생성
- [ ] QR 스캔
- [ ] 6자리 코드
- [ ] 연결 상태
- [ ] PC -> phone
- [ ] phone -> PC
- [ ] 수락/거절
- [ ] 다운로드/저장
- [ ] 파일 크기
- [ ] 기타
