# Support FAQ

## 접속 주소가 뭔가요?

`https://app.getvibeshare.com`입니다. 일반 사용자는 이 주소만 알면 됩니다.

## 서버 주소도 입력해야 하나요?

아니요. `https://api.getvibeshare.com`은 운영용 API 주소입니다. 일반 사용자에게 안내할 필요가 없습니다.

## PC에 카메라가 필요한가요?

아니요. PC는 QR을 보여주기만 합니다. 휴대폰 카메라가 PC 화면의 QR을 스캔합니다.

## 앱 설치가 필요한가요?

현재 공개 기본 버전은 web-first입니다. PC와 휴대폰에서 웹으로 사용합니다.

## 사용 방법은 무엇인가요?

1. PC에서 `https://app.getvibeshare.com`을 엽니다.
2. 휴대폰 카메라로 QR을 스캔합니다.
3. 연결 후 보낼지 받을지 선택합니다.
4. 파일을 보내고 받는 쪽에서 수락합니다.

## QR이 안 됩니다.

PC 화면에서 `새 QR 만들기`를 누른 뒤 다시 스캔해 주세요. 계속 안 되면 6자리 코드를 입력해 주세요.

## 6자리 코드는 어디에 쓰나요?

QR 스캔이 어려울 때 수동으로 같은 세션에 들어가는 fallback입니다.

## 파일은 자동으로 저장되나요?

아니요. 받는 쪽이 수락해야 하고, 다운로드 버튼을 눌러 저장합니다.

## 다운로드한 파일은 어디에 있나요?

PC는 브라우저의 기본 다운로드 폴더를 확인하세요. iPhone은 Safari 다운로드 또는 파일 앱 Downloads 폴더를 확인하세요.

## 파일이 계속 저장되나요?

아니요. Vibe Share는 전송을 위한 임시 저장을 기준으로 합니다. 영구 보관 서비스가 아닙니다.

## 큰 파일도 되나요?

운영 설정에는 큰 파일을 위한 제한값이 준비되어 있지만, 실제 성공 여부는 브라우저, 네트워크, 운영 비용 제한의 영향을 받습니다. 베타에서는 중요한 대용량 파일을 유일한 원본으로 보내지 마세요.

## 보안상 안전한가요?

받는 쪽이 파일 이름과 크기를 확인하고 수락/거절할 수 있습니다. 다만 "완전 안전", "해킹 불가", "모든 파일 자동 악성코드 검사 완료"를 약속하지 않습니다.

## 문제가 생기면 무엇을 보내야 하나요?

- PC 브라우저
- 휴대폰 모델과 OS
- 전송 방향
- 파일 종류와 크기
- QR 또는 6자리 코드 사용 여부
- 오류 문구
- 가능하면 스크린샷
