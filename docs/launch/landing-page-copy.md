QR로 PC와 휴대폰을 연결해 파일을 바로 주고받는 웹 서비스.

```text

```


## 핵심 기능

- QR로 PC와 휴대폰 연결
- 6자리 코드 수동 연결
- PC -> phone 전송
- phone -> PC 전송
- 받는 쪽 수락/거절과 다운로드

## 왜 편한가요

- 케이블을 찾지 않아도 됩니다.
- 메신저나 이메일로 자기 자신에게 보내지 않아도 됩니다.
- PC에 카메라가 없어도 됩니다.
- 같은 연결에서 양쪽 방향 전송을 모두 처리합니다.
- 받기 전에 파일 이름과 크기를 확인할 수 있습니다.

## 사용 방법

1. PC에서 `https://app.getvibeshare.com`을 엽니다.
2. 휴대폰 카메라로 PC 화면의 QR을 스캔합니다.
3. 연결 후 보낼지 받을지 선택합니다.
4. 파일을 보내고 받는 쪽에서 수락합니다.

## FAQ 짧은 문구

### PC 카메라가 필요한가요?

아니요. PC는 QR을 보여주기만 합니다.

### 앱 설치가 필요한가요?

현재 공개 기본 버전은 web-first입니다. PC와 휴대폰 브라우저로 바로 시작합니다.

### 서버 주소도 알아야 하나요?

아니요. 사용자는 `https://app.getvibeshare.com`만 열면 됩니다.

### 파일은 계속 저장되나요?

아니요. 전송을 위한 임시 저장을 기준으로 운영합니다.
