# Executive Summary

## 제품

Vibe Share는 QR로 PC와 휴대폰을 연결해 파일을 양방향으로 주고받는 웹 서비스입니다.

## 현재 상태

공개 web-first 배포가 완료되어 실제 사용자는 아래 주소로 시작할 수 있습니다.

```text
https://app.getvibeshare.com
```

운영 API는 아래 주소에서 확인합니다.

```text
https://api.getvibeshare.com
```

## 완료된 핵심 기능

- PC 웹 세션 생성
- QR 표시
- 6자리 코드 표시
- 휴대폰 QR 스캔 연결
- 6자리 코드 수동 연결
- PC -> phone 전송
- phone -> PC 전송
- 받는 쪽 수락/거절
- 진행 상태와 성공/실패 상태
- 파일 메타데이터 표시
- 다운로드/저장 액션

## 완료된 운영 기반

- Railway API

- Railway Redis
- Cloudflare Pages
