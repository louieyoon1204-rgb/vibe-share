Web-first file transfer

PC와 휴대폰, QR로 바로 연결하세요.

Vibe Share는 설치 없이 웹에서 시작하는 파일 전송 서비스입니다. PC에서 QR을 보여주고 휴대폰이 스캔하면, 같은 연결 안에서 파일을 양방향으로 보낼 수 있습니다.

Vibe Share 열기 사용 방법

핵심 기능

PC 카메라 불필요

PC는 QR을 보여주고 휴대폰이 스캔합니다.

양방향 전송

PC에서 휴대폰으로, 휴대폰에서 PC로 보낼 수 있습니다.

수락/거절

받는 쪽이 파일 이름과 크기를 확인한 뒤 선택합니다.

6자리 코드

QR이 어려울 때 수동 코드로 연결합니다.

다운로드 액션

수락 후 다운로드 버튼으로 저장합니다.

운영 확인

운영자는 health, 로그, 저장소 상태를 확인합니다.

사용 방법

  1. PC에서 https://app.getvibeshare.com을 엽니다.
  2. 휴대폰 카메라로 PC 화면의 QR을 스캔합니다.
  3. 연결 후 보낼지 받을지 선택합니다.
  4. 파일을 보내고 받는 쪽에서 수락합니다.

베타 안내

현재 공개 기본 버전은 web-first입니다. 파일은 전송을 위한 임시 저장 기준으로 처리되며, 영구 보관이나 무제한 전송을 약속하지 않습니다.

FAQ

PC 카메라가 필요한가요?
아니요. PC는 QR을 보여주기만 합니다.

설치가 필요한가요?
현재 공개 기본 버전은 웹에서 바로 사용합니다.

서버 주소도 알아야 하나요?
아니요. 사용자는 https://app.getvibeshare.com 만 열면 됩니다.

파일이 계속 저장되나요?
아니요. 전송을 위한 임시 저장을 기준으로 운영합니다.
