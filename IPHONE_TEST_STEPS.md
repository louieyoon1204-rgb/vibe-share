# iPhone Test Steps

이 문서는 iPhone 테스터에게 그대로 보낼 수 있는 공개 web-first 테스트 안내입니다.

## 테스트 주소

```text
https://app.getvibeshare.com
```

서버 주소는 일반 사용자에게 필요하지 않습니다.

## 기본 테스트

1. PC에서 `https://app.getvibeshare.com`을 엽니다.
2. PC 화면에 QR과 6자리 코드가 보이는지 확인합니다.
3. iPhone 기본 카메라로 PC 화면의 QR을 스캔합니다.
4. Safari가 열리면 연결 상태가 `연결됨`으로 바뀌는지 확인합니다.
5. 연결 후 PC와 iPhone 양쪽에서 보낼지/받을지 선택합니다.

## PC -> iPhone

1. PC에서 `휴대폰으로 파일 보내기`를 누릅니다.
2. 작은 이미지나 문서 파일을 선택합니다.
3. iPhone 화면에서 파일 이름, 크기, 상태를 확인합니다.
4. `수락`을 누릅니다.
5. `다운로드`를 눌러 저장합니다.
6. Safari 다운로드 또는 파일 앱 Downloads 폴더에서 확인합니다.

## iPhone -> PC

1. iPhone 화면에서 `PC로 파일 보내기`를 누릅니다.
2. 사진이나 파일을 선택합니다.
3. PC 화면에서 파일 이름, 크기, 상태를 확인합니다.
4. `수락`을 누릅니다.
5. PC에서 다운로드합니다.

## QR이 안 될 때

1. PC 화면의 6자리 코드를 확인합니다.
2. iPhone 화면에서 수동 코드 입력을 엽니다.
3. 6자리 코드를 입력합니다.
4. 그래도 안 되면 PC에서 `새 QR 만들기`를 누르고 다시 스캔합니다.

## 확인해 줄 것

- QR 스캔이 쉬운가요?
- 연결됨 상태가 명확한가요?
- PC -> iPhone 전송이 이해되나요?
- iPhone -> PC 전송이 이해되나요?
- 수락/거절 버튼이 헷갈리지 않나요?
- 다운로드 위치를 찾을 수 있나요?
- 오류 메시지가 한국어로 이해되나요?
